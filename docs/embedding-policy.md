# 埋め込み・引用・サムネイル ポリシー

このサービスで**最も重要な文書**。記事を1本書く前に、掲載する各動画をこのフローに通す。
1つでもNGに当たったら **掲載しない**。迷ったら載せない。

> 原則: 我々は動画を「持たない・配らない」。**公式が公開している場所へ案内する**だけ。

---

## 0. 大前提（4つの絶対禁止）

これに該当する行為は理由を問わず一切しない。

- ❌ 動画ファイルをダウンロードして再アップロードする
- ❌ 試合映像を切り抜いて自サイトで配信する
- ❌ 非公式・転載・権利者不明の動画を埋め込む／リンクする
- ❌ ジオブロック回避・有料配信回避・「無料で見る裏技」を案内する

---

## 1. 掲載可否フローチャート

各動画について上から順に判定する。

```
START
 │
 ├─ Q1. 公式（権利元・リーグ・クラブ・放送局・配信権者）が公開した動画か？
 │       NO  → ❌ 掲載しない（is_official=false は原則扱わない）
 │       YES ↓
 │
 ├─ Q2. 権利者・情報源を特定できるか？（source_name / rights_holder_type が埋まるか）
 │       NO  → ❌ 掲載しない（権利者不明は扱わない）
 │       YES ↓
 │
 ├─ Q3. 掲載形態はどれか？
 │       (A) 公式ページへのリンクのみ         → ✅ 常にOK（最も安全）
 │       (B) 公式が提供する埋め込みコードを使う → Q4へ
 │       (C) それ以外（自前プレイヤー等）       → ❌ 掲載しない
 │
 ├─ Q4. 公式の埋め込みコード/プレイヤーをそのまま使うか？
 │       （改変・広告除去・プレイヤー隠しをしない）
 │       NO  → ❌ 埋め込まない（リンクのみに切替→A扱い）
 │       YES ↓
 │
 ├─ Q5. その動画ページ/プラットフォームの規約が埋め込みを許可しているか？
 │       不明/禁止 → ❌ 埋め込まない（リンクのみに切替→A扱い）
 │       許可       → ✅ 埋め込みOK（is_embed_allowed=true）
 │
 └─ END
```

**実務上の既定**: 判断に迷うソースは **(A) リンクのみ** にフォールバックする。リンクは常に安全。

## 2. ソース種別ごとの初期ルール

| ソース | リンク | 埋め込み | 備考 |
|---|---|---|---|
| 公式YouTubeチャンネルの動画 | ✅ | ✅（YouTube公式iframe） | 埋め込み無効化されていれば**リンクのみ**。チャンネルが公式本物かを必ず確認 |
| FIFA+ / UEFA.tv 等の公式配信ページ | ✅ | △ | 公式埋め込みコードが提供されている時のみ埋め込み。無ければリンクのみ |
| リーグ公式（Premier League Video 等） | ✅ | △ | 同上 |
| クラブ公式サイト | ✅ | △ | 同上 |
| 放送局・配信権者公式 | ✅ | △ | 同上。有料領域への送客は「公式配信」として明示 |
| 非公式・転載・切り抜き | ❌ | ❌ | 一切扱わない |
| 権利者不明 | ❌ | ❌ | 一切扱わない |

△ = 公式が埋め込みコードを提供している場合のみ可。それ以外はリンクのみ。

## 2.5 埋め込み可否の判定手順（YouTube）

埋め込みは**公式の方針**。ただし「公式に見える動画」の多くは各国放送局の地域限定アップロードで、
**埋め込めても日本からは再生できない**。埋め込む前に必ず次を確認する。

### 手順

1. **チャンネルを確定する（タイトルで判断しない）**
   タイトルは各放送局が酷似した名前を付けるため当てにならない（例: 「Match Highlights | FIFA World Cup 2026™」は
   TVNZ / SuperSport / Mediacorp など複数の放送局が使用）。
   YouTube oEmbed で発行元チャンネルを機械的に確認する:
   ```
   https://www.youtube.com/oembed?url=https://www.youtube.com/watch?v=<VIDEO_ID>&format=json
   → 返ってくる "author_name" を見る
   ```
2. **`author_name` が `FIFA`（または対象国の正規配信権者）なら採用候補。** それ以外の放送局名なら地域限定の可能性が高い → 不採用。
   - `FIFA` → 全世界配信。日本からも視聴可。**最優先で採用。**
   - 放送局名（FOX / TSN / ITV / SuperSport / TVNZ / Mediacorp / tod など）→ その国/地域限定。日本不可なら不採用。
3. **埋め込み許可を「実際に埋め込んで」確認する。** ⚠️ oEmbed が成功してもチャンネルが分かるだけで、**埋め込み可否は分からない**。
   所有者が埋め込みを無効化していると、iframe 再生時に **エラー 150 / 153（「動画の所有者が外部サイトでの再生を無効にしています」）** が出る。
   - 実例: FIFA公式チャンネルのW杯ハイライトは **author_name=FIFA だが埋め込み無効（エラー153）**。→ 埋め込めない。
   - 判定: 実機の iframe で正常再生 → `is_embed_allowed=true`。エラー150/153 → `is_embed_allowed=false` → **リンクのみ**に切替。
   - **権利元が埋め込みを禁止しているなら、それを尊重する。**「公式だけ・権利尊重」がこのサービスの本質。回避しない。
4. **公開前に実機で日本から再生確認する。** ジオブロックも埋め込み設定も後から変わるため `last_checked_at` を更新する。

> まとめ: 埋め込みは「できる時だけ」やる。FIFAのように所有者が無効化している場合は、映像を抱え込まず**公式ページへ送客するリンクカード**で案内する（これも仕様書OKリストの「公式ページへのリンク」）。

### 検証済みの実例（2026-06-22 実機レンダリングで確認）

iframe を実際に描画し、「error 153 / ブロック表示」か「正常再生」かで判定した結果:

| ソース | チャンネル(author) | 埋め込み | 扱い |
|---|---|---|---|
| FIFA（W杯ハイライト） | FIFA | ❌ ブロック（「このウェブサイトでの表示はブロックされています」） | リンクのみ |
| DAZN（Jリーグ公式ハイライト） | DAZN Japan | ✅ 正常再生 | **埋め込み採用** |
| Premier League（公式ハイライト） | Premier League | ✅ 正常再生 | **埋め込み採用** |
| 各国放送局（FOX/TSN/ITV/SuperSport/TVNZ/Mediacorp/tod等） | 各局 | （地域限定） | 日本不可なら不採用 |

> 重要: **DAZNのコンテンツはYouTube（DAZN Japanチャンネル）経由で埋め込み可能**。DAZNサイト/アプリ自体の埋め込みコードは提供されないため、リンク（フルマッチ送客）で扱う。

### 多重ソース方針（常に適用）

1試合につき**複数の公式ソースを並列に配置**する。各ソースに `is_embed_allowed` / `geo_available_japan` を持たせ、
埋め込み可かつ日本可のものを優先して埋め込み、それ以外は公式リンクとして並べる。
こうすることで「埋め込み不可・地域限定・削除」のいずれが起きても、別の公式ソースで視聴導線を確保できる。

### 採用する埋め込みコード

`youtube-nocookie.com`（プライバシー強化モード）を使う。プレイヤー・広告・公式表記は一切改変しない。
```html
<iframe width="560" height="315"
  src="https://www.youtube-nocookie.com/embed/<VIDEO_ID>"
  title="<公式タイトルを引用>"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen></iframe>
```

> 広告について: 公式プレイヤー内に出る広告は権利元の収益であり**除去・隠蔽しない**。
> 当サイト自体は広告ゼロにすることで、雑多な転載サイトより快適な「公式への入口」を提供する。

## 3. サムネイル・画像ポリシー

サムネは権利が重い領域。**自前保存・再配信は原則しない。**

- ✅ OK: 公式の OGP / oEmbed が返す画像を、**公式が許容する形で**カード表示（ホットリンクは相手規約次第なので慎重に）
- ✅ OK: 自分で撮った画像・自作の図・テキストのみのカード
- ❌ NG: 公式サムネを自サーバへ保存して再配信する
- ❌ NG: サムネを加工・トリミングして独自素材化する
- ⚠️ 迷う場合: **画像を出さず、テキストカード＋公式リンク**にする（ネタバレ防止とも相性が良い）

YouTube埋め込みは iframe 内で公式サムネ・プレイヤー・広告が表示されるため、**iframe をそのまま使う限り**この問題は起きない。サムネ単体の切り出し保存だけが危険。

---

## 4. テキスト（紹介文・引用）ポリシー

- ✅ OK: 公式タイトルの**引用**（出典明示・短く）
- ✅ OK: 自分で書いた1〜2行の日本語紹介・見どころ
- ✅ OK: 公式APIが返すメタ情報（尺・日付・大会名など）の表示
- ❌ NG: 動画内容の全文文字起こし・全訳の転載
- ❌ NG: 公式の説明文を丸ごとコピペして自記事化する

紹介文は「**短い案内**」に留める。中身を代替するほど書かない。

---

## 5. 取得・巡回ポリシー（Phase 3で関係）

- ✅ OK: 公式RSS / サイトマップ / YouTube Data API / 公式API があるものだけ自動取得
- ❌ NG: スクレイピング禁止サイトの無理な巡回、robots.txt 無視、過負荷アクセス
- 手動キュレーション期（Phase 1〜2）は人間が公式ページを見て記録する。自動化はAPI/RSSが整ってから。

---

## 6. 収益化と権利の関係（注意）

- 収益化すると権利者の見る目が厳しくなる。**映像そのものを収益の主コンテンツにしない。**
- アフィリエイトは公式配信サービス・グッズ等へ。**VPN等ジオブロック回避商材は扱わない。**
- 公式プレイヤー内の広告を隠さない・権利元への送客導線を妨げない。

---

## 7. 削除・権利申し立て対応

- 連絡先（問い合わせ窓口）を常設する。
- 権利者から要請があれば**即時に該当掲載を取り下げる**運用にする。
- 取り下げたレコードは削除せず `status=removed` にして履歴を残す（再発防止）。
